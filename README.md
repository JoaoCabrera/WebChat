## Para rodar o projeto:

É necessario ter o Git, Node e MongoDB instalados para rodar o projeto.

## Clone o projeto:

`git clone git@github.com:JoaoCabrera/WebChat.git`

## Instale as dependências:

`npm install`

## Inicie o projeto:

`npm start`
